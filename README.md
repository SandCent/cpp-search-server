# C++ Search Server

Реализация поискового сервера с поддержкой TF-IDF.

## Описание
Проект реализует систему поиска документов по текстовым запросам.

Поддерживает:
- индексацию документов
- поиск по ключевым словам
- фильтрацию по статусу
- сортировку по релевантности и рейтингу

## ⚙️ Технологии
- C++
- STL (std::map, std::set, std::vector)
- Алгоритмы TF-IDF

## Запуск
Скомпилировать:

g++ main.cpp -o search
